# Movie Roulette App

## Runing App

1. Clone repository
2. npm i
3. npm start
4. You must have the account on https://www.themoviedb.org/ in order to login

## Features

1. Random movies suggestion - most popular
2. Suggest most popular movies based on chosen genre
3. Checks movie details
